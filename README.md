### 필요

---
- 포트원 관리자에서 확인이 가능한 api-key, secret-key 정보를 담은 properties 파일 필요.
    ```properties
    key.api={값입력}
    key.secret={값입력}
    ```
- localhost 테스트 시 웹훅을 위한 포트포워딩 프로그램
  - [ngrok](https://ngrok.com/)
